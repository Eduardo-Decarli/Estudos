# Conceito

O HTML é uma linguagem de marcação de texto, comumente utilizada para realizar indexação de texto na rede da internet. Ele permite: 

- Estruturar texto através de tags
- Realizar indexação entre páginas através do hypertext
- Organizar código semanticamente legível
- Auxiliar na acessibilidade de leitura de páginas

## Como funciona a WEB

Ela funciona respectivamente como uma teia de informações, onde nós enviam solicitações e respostas para outros nós, atualmente a Web é feita em modelo cliente-servidor, onde um cliente realiza uma solicitação para o servidor (um nó) e recebe a informação necessária. Essa solicitação é feita utilizando um protocolo chamado HTTP, que serve para padronizar as respostas que trafegam pela internet.

O trafego na internet funciona mais ou menos da seguinte forma: 

1. Computador solicita informação para um servidor via url (www.google.com)
2. A solicitação é interceptada pelo Servidor DNS do seu provedor de internet
3. O Servidor DNS traduzirá a URL para um endereço IP Público.
4. A solicitação então é enviada via HTTP para o endereço IP Público traduzido através da URL.
5. O servidor final recebe e processa a mensagem
6. O servidor final envia os arquivos e dados solicitados
7. Um arquivo HTML é recebido e traduzido pelo navegador do usuário.

## Conceitos Importantes

**Internet:** Inter (entre) e net(rede), ou seja, representa o termo **entre redes** de computadores. Abstratamente falando, seria uma teia que conecta os computadores do mundo.

**Protocolos (HTTP/HTTPS):** São padrões de transmissão de dados, um protocolo serve exclusivamente para padronizar envio de informações, como a computação é sensível até a mínima alteração de fluxo, é necessário criar padrões de comportamento para que computadores consigam conversar, onde um fala e o outro entenda.

**HTTP:** É considerado o principal protocolo da Web, é por meio desse protocolo que as informações trafegam, ele possui uma estrutura de cabeçalho (metadados), corpo, método, status, URL e versão do HTTP
