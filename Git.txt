# GIT
Git é um sistema de controle de versão distribuído que permite rastrear 
alterações em arquivos e coordenar o trabalho em projetos entre várias pessoas.
Ele armazena versões do código em um repositório local, permitindo voltar a versões 
anteriores, criar ramificações independentes para novos recursos, mesclar alterações 
e colaborar com eficiência em equipes.

# GITHUB

GitHub é uma plataforma online que hospeda repositórios Git, permitindo que desenvolvedores 
armazenem e compartilhem seus repositórios na nuvem.Além de fornecer controle de versão, o 
GitHub oferece ferramentas de colaboração, como revisão de código, relatórios de problemas (issues), 
integração contínua e implantação (CI/CD), e uma interface gráfica que facilita o uso do Git.

# GitBash

Ao instalar o Git no dispositivo, ele oferece a instalação do GitBash, O Git Bash é um aplicativo 
que oferece uma interface de linha de comando (CLI) para usar os comandos do Git
Além dos comandos Git, o Git Bash também fornece várias ferramentas e comandos Unix, como ls, pwd, ssh, e grep, 
facilitando a interação com o Git de uma maneira mais eficiente no Windows.
